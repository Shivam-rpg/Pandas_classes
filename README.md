# 🐼 Pandas Learning Series (Class 1–9)

This repository contains a **beginner-friendly Pandas learning series** implemented using Jupyter Notebooks.
It covers the **fundamentals of data analysis using Python and Pandas**, starting from basic data structures to filtering, sorting, and statistics.

The notebooks are designed for **students and beginners** who want to understand how to work with tabular data efficiently in Python.

---

# 📚 Course Contents

## 1️⃣ Pandas Class 1 – Introduction & Setup

Topics covered:

* What is **Pandas** and why it is used
* Installing Pandas
* Importing Pandas and NumPy
* Understanding **Series**
* Understanding **DataFrame**
* Creating Series from:

  * Lists
  * Dictionaries
* Creating DataFrames from:

  * Dictionaries
  * NumPy arrays


## 2️⃣ Pandas Class 2 – DataFrame Basics

Topics covered:

* Inspecting data

  * `head()`
  * `tail()`
  * `info()`
  * `shape`
  * `dtypes`
* Selecting columns and rows
* Using:

  * `loc`
  * `iloc`
* Understanding:

  * `index`
  * `columns`
  * `values`
* Renaming columns
* Setting and resetting index


## 3️⃣ Pandas Class 3 – Data Cleaning

Topics covered:

* Handling missing values

  * `isnull()`
  * `isna()`
  * `dropna()`
  * `fillna()`
* Replacing missing values using:

  * Mean
  * Mode
* Changing data types using:

  * `astype()`
* Checking dataset structure with `info()`


## 4️⃣ Pandas Class 4 – Data Selection & Filtering

Topics covered:

* Boolean indexing
* Conditional selection
* Filtering rows using conditions
* Combining conditions with:

  * `&` (AND)
  * `|` (OR)
  * `~` (NOT)

## 5️⃣ Pandas Class 5 – Sorting & Statistics

Topics covered:

* Sorting data using:

  * `sort_values()`
  * `sort_index()`
* Sorting multiple columns
* Descriptive statistics:

  * `mean()`
  * `median()`
  * `mode()`
  * `std()`
  * `describe()`
 
## 6️⃣ Pandas Class 6 – GroupBy & Aggregations

Topics covered:

Understanding the Split–Apply–Combine concept
Grouping data using groupby()
* Aggregation functions:
  * `sum()`
  * `mean()`
  * `count()`
  * `min()`
  * `max()`

* Advanced aggregation:

Using `agg()` for multiple aggregations
Applying different functions to different columns


## 7️⃣ Pandas Class 7 – Merging, Joining & Concatenation

Topics covered:

* Combining datasets using:

  * `pd.concat()`
  * `Vertical stacking (axis=0)`
  * `Horizontal stacking (axis=1)`

* SQL-style joins using:

  * `merge()`
  * `inner join`
  * `left join`
  * `right join`
  * `outer join`

* Working with indexes:

Using join() to combine DataFrames

## 8️⃣ Pandas Class 8 – Working with Time Series

Topics covered:

* Handling date and time data:

  * `Converting columns to datetime using pd.to_datetime()`

* Time-based indexing:

  * `Setting DateTime index using set_index()`

* Time series analysis:

  * `Resampling data (resample())`

* Aggregating by time periods such as:
  * `Daily (D)`
  *` Weekly (W)`
  *` Monthly (M)`
  * `Yearly (Y)`

* Additional operations:

  * `Date-based filtering`
  * `Rolling window calculations`
  * `Expanding window statistics`


## 9️⃣ Pandas Class 9 – Advanced Pandas Techniques

Topics covered:

* Applying custom transformations:

  * `apply() for row/column operations`
  * `map() for element-wise transformations`

* Advanced data manipulation techniques:

  * `Custom function transformations`
  * `Efficient data processing`
  * `Writing cleaner and optimized Pandas code`
