# 🐼 Pandas Learning Series (Class 1–9)

This repository contains a **beginner-friendly Pandas learning series** implemented using Jupyter Notebooks.
It covers the **fundamentals of data analysis using Python and Pandas**, starting from basic data structures to filtering, sorting, and statistics.

The notebooks are designed for **students and beginners** who want to understand how to work with tabular data efficiently in Python.

---

# 📚 Course Contents

## 1️⃣ Pandas Class 1 – Introduction & Setup

Topics covered:

* What is **Pandas** and why it is used
* Installing Pandas
* Importing Pandas and NumPy
* Understanding **Series**
* Understanding **DataFrame**
* Creating Series from:

  * Lists
  * Dictionaries
* Creating DataFrames from:

  * Dictionaries
  * NumPy arrays


## 2️⃣ Pandas Class 2 – DataFrame Basics

Topics covered:

* Inspecting data

  * `head()`
  * `tail()`
  * `info()`
  * `shape`
  * `dtypes`
* Selecting columns and rows
* Using:

  * `loc`
  * `iloc`
* Understanding:

  * `index`
  * `columns`
  * `values`
* Renaming columns
* Setting and resetting index


## 3️⃣ Pandas Class 3 – Data Cleaning

Topics covered:

* Handling missing values

  * `isnull()`
  * `isna()`
  * `dropna()`
  * `fillna()`
* Replacing missing values using:

  * Mean
  * Mode
* Changing data types using:

  * `astype()`
* Checking dataset structure with `info()`


## 4️⃣ Pandas Class 4 – Data Selection & Filtering

Topics covered:

* Boolean indexing
* Conditional selection
* Filtering rows using conditions
* Combining conditions with:

  * `&` (AND)
  * `|` (OR)
  * `~` (NOT)

## 5️⃣ Pandas Class 5 – Sorting & Statistics

Topics covered:

* Sorting data using:

  * `sort_values()`
  * `sort_index()`
* Sorting multiple columns
* Descriptive statistics:

  * `mean()`
  * `median()`
  * `mode()`
  * `std()`
  * `describe()`

# 🛠️ Technologies Used

* **Python**
* **Pandas**
* **NumPy**
* **Jupyter Notebook**
